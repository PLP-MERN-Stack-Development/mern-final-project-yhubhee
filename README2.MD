A complete social media app built from scratch!

## Features
- Real-time posts with images
- Likes & comments (instant)
- Private messaging with online status
- Avatar upload
- Beautiful profile page
- Dark mode + Mobile responsive
- PWA ready

## Tech Stack
- Frontend: React + Vite + Tailwind + Socket.io
- Backend: Node.js + Express + MongoDB (native driver) + Socket.io
- Auth: JWT
- Real-time: Socket.io
- Deployment ready
